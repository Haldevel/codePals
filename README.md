## Prerequisites
<hr />
What things you need installed before running this application.

    npm install 
    npm install sequelize 
    npm install mysql2
    npm install express 
    npm install express-handlebars 
    npm install passport
    npm install bcryptjs
    -- npm install dotenv --save
